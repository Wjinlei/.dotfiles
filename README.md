### 介绍
这是我的各种配置文件.<br/>
它使用`rcm`来管理<br/>

### 使用前,您需要
- [ ] 安装`rcm`

### 快速开始
```bash
rcup -t zsh # 恢复zsh相关的配置
rcup -t ranger # 恢复ranger相关的配置
...
```
